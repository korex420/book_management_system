# Book Management System

A complete PHP/MySQL web application for managing book collections with CRUD operations and advanced search capabilities.

## Features

- ✅ Create, Read, Update, Delete books
- ✅ Advanced search with multiple criteria
- ✅ AJAX autocomplete for authors and genres
- ✅ Security protection (SQL Injection, XSS, CSRF)
- ✅ Responsive Bootstrap design
- ✅ Template engine separation

## Installation

1. Upload all files to your web server
2. Create MySQL database using `database_setup.sql`
3. Update database credentials in `config/config.php`
4. Ensure PHP PDO MySQL extension is enabled
